## Exercises
1. Write a JavaScript program to list the properties of a JavaScript object. 

Sample object:
```javascript 
var student = {
  name : "David Rayy",
  sclass : "VI",
  rollno : 12 
};
```
Sample Output:
```
name,sclass,rollno
```
